# vuecli-uliweb-example

## Directory structure

```
.
├── babel.config.js
├── backend //Uliweb back-end project
├── devproxy //Proxy for combine front-end and back-end dev servers
├── node_modules
├── package.json
├── package-lock.json
├── public
├── README.md
└── src
```

npm install
